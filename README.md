# Play Nirvana Games

![Branding](https://github.com/kniffen/Random-Outcomes/blob/master/branding/logo_feature.png)

### A mobile app with a collection of randomized outcomes including:
- Dice roll
- Cards
- Coin flip
- 8 ball
- Spin the bottle
- Pick a straw
- Rock, Paper, Scissors
- Roulette
- Spin the wheel
- Random number generator
- Three cups
- Russian roulette
